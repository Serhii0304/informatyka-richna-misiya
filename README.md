# Інформатика: річна місія

Готовий сайт онлайн-тестів для екстернів з інформатики.

## Що завантажувати на GitHub

Завантажуйте весь вміст цієї папки:

- `index.html`
- `styles.css`
- `app.js`
- `admin.html`
- `admin.css`
- `admin.js`
- `server.py`
- `Запустити сайт.bat`
- папку `assets`

## Запуск

Для локального запуску відкрийте файл `Запустити сайт.bat`.

Після запуску сервер покаже адресу сайту. Зазвичай це:

`http://127.0.0.1:8765/`

## Важливо

IP-блокування повторного проходження та серверне збереження результатів працюють через `server.py`.

GitHub Pages не запускає Python-сервер, тому для повної роботи з IP-блокуванням потрібен хостинг, який підтримує Python, або локальний запуск через `Запустити сайт.bat`.
